## JulietCafe – Food Delivery Web Application

JulietCafe is a modern **frontend food delivery web application** built using React.  
Users can explore dishes, add items to their cart, and place orders through a simple and responsive interface.

---

### 🚀 Features

- **Explore Our Menu** – Browse different food items available on the platform  
- **Top Dishes Near You** – Highlights popular dishes for users  
- **Authentication** – Login and Signup forms for users  
- **Cart Functionality** – Add and remove food items from the cart  
- **Place Order Page** – Review cart items and place orders  
- **Responsive Design** – Optimized for desktop and mobile devices  

---

### 🛠 Tech Stack

- **JavaScript**
